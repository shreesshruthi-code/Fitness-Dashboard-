Project Name:Fitness Progress Dashboard

Intern ID:CITS2792

Description

This project is a Fitness Progress Dashboard developed using HTML and CSS. It provides a modern and user-friendly interface to track daily workouts, diet plans, alarms, weekly schedules, and fitness progress.

Features

- Responsive design
- Dashboard layout with sidebar and navigation bar
- Today's workout plan
- Diet schedule
- Alarm and reminder section
- Weekly workout planner
- Fitness statistics and progress cards
- Clean and modern user interface

Technologies Used

- HTML
- CSS

Files Included

- index.html
- style.css
- Images
- Screenshots
